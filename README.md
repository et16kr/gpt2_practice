# gpt2_practice

GPT2 구현 연습용 repo입니다.

- Hugging Face 모델 경로: https://huggingface.co/openai-community/gpt2
- 목표: `config.json`을 기준으로 GPT-2 구조를 이해하고 직접 구현/실험하기
- 토크나이저 사용 가이드: `REQUIREMENTS.md`의 `12. 토크나이저 사용 가이드 (초심자용)`
- 참고 샘플 코드: `examples/` (토크나이저, for-loop GPT-2 연산, 토크나이저 입력셋, Python -> C++/CUDA 연동)
- 모델 파일 파싱 설계: `REQUIREMENTS.md`의 `13. model.safetensors 파싱 설계 (C++ 기준)`
